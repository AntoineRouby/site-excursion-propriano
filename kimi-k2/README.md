# Écho des Vagues - Application Web

## Installation Windows (Sans admin)

### Prérequis
- Node.js : télécharger depuis nodejs.org
- Git : télécharger depuis git-scm.com

### Installation
1. Ouvrir PowerShell
2. Naviguer dans ce dossier
3. Suivre les instructions

### Commandes PowerShell utiles
- cd : changer de dossier
- ls ou dir : lister les fichiers
- mkdir : créer un dossier
- echo : afficher du texte
